# 投资看板 - 实时数据版

## 更新说明 (2026-04-22)

### 改造内容

本版本将投资看板从静态数据更新模式改造为前端直接调用第三方API获取实时数据。

### 主要变更

1. **新增 api-service.js**
   - 集成天天基金实时净值API (JSONP格式，支持跨域)
   - 数据源：`http://fundgz.1234567.com.cn/js/{基金代码}.js`
   - 支持批量获取基金实时估值数据
   - 内置5分钟缓存机制，避免频繁请求

2. **更新 app.js**
   - 页面加载时自动获取实时基金数据
   - 支持手动刷新和自动刷新功能
   - 保留静态数据作为后备方案
   - 实时估值与静态净值双显示

3. **更新 index.html**
   - 添加API服务脚本引用
   - 更新头部显示"实时数据来源：天天基金"
   - 基金详情模态框增加"实时估值"字段

### 技术说明

天天基金API返回格式：
```json
jsonpgz({
  "fundcode":"510300",
  "name":"沪深300ETF华泰柏瑞",
  "jzrq":"2026-04-21",
  "dwjz":"4.7789",
  "gsz":"4.7931",
  "gszzl":"0.30",
  "gztime":"2026-04-22 11:30"
});
```

字段说明：
- `fundcode`: 基金代码
- `name`: 基金名称
- `jzrq`: 净值日期
- `dwjz`: 单位净值
- `gsz`: 估算净值（实时估值）
- `gszzl`: 估算涨跌幅
- `gztime`: 估值时间

### 注意事项

1. 基金实时估值数据在工作日9:30-15:00期间更新
2. 周末及节假日估值数据可能不更新
3. 实时估值仅供参考，实际净值以基金公司公布为准
4. 请合理设置刷新间隔，避免请求过于频繁
